What is CogNexus

CogNexus is a distributed cognitive hub that enables AI Agent capabilities to flow and exchange. You can discover other people's Agents here, or publish your own for others to use.

💡

CogNexus is currently in experimental stage. Features and APIs may change at any time.

Agent Types

CogNexus supports three types of Agents:

👤 Human

A real user's knowledge base. Built with Cogmate, it contains the user's perspectives, experiences, and expertise. You can ask a Human Agent questions and receive answers based on their knowledge base.

🎭 Character (Coming Soon)

AI role-playing. Can be historical figures, fictional characters, or domain experts. Chat with a Character to experience different perspectives and ways of thinking.

🔮 Simulation (Coming Soon)

Simulated environments and scenarios. Used for testing decisions, rehearsing conversations, and exploring hypothetical situations.

ATP Credits

ATP (Agent Trade Points) is CogNexus's internal credit system used for exchanging Agent capabilities.

Method Description
Registration Bonus New users receive 100 ATP
Sell Tokens Earned when others purchase your Agent Tokens
⚠️

Important: ATP is an internal platform credit only. It has no real-world monetary value and cannot be redeemed, withdrawn, or transferred.

Token Mechanism

Tokens are credentials for accessing Agents. Different token types have different permissions:

Type Permission Description
full Full Access Browse knowledge base + Q&A interaction + View details
qa_public Public Q&A Limited Q&A sessions, suitable for trials
browse_public Browse Only Can only view public content, no interaction

Tokens typically have an expiration date and need to be repurchased after expiry.

Browse Market

On the Market page you can see all published Agents. Each Agent card displays:

  • Name and description — basic Agent information
  • Type tag — Human / Character / Simulation
  • Price — ATP required to purchase a Token
  • Online status — whether the Agent is accessible

Purchase Token

After finding an Agent you're interested in:

  1. Click the "Purchase" button on the Agent card
  2. Confirm the price and Token type
  3. ATP is deducted from your balance
  4. Receive your Token, viewable in the Dashboard

Use Agent

After purchasing a Token, you can find it in the "Purchased" section of your Dashboard.

  1. Click "Copy" to get the full Token
  2. Click "Visit" to go to the Agent interface
  3. Use the Token in the Agent interface to interact

Publish Agent

To publish your own Agent, you first need to deploy Cogmate as the Agent backend.

Once deployment is complete:

  1. Click "Publish Agent" in the Dashboard
  2. Enter your Cogmate API address
  3. Click "Detect" to auto-fetch Agent information
  4. Add description and pricing
  5. Submit for publication

Manage Tokens

After publishing an Agent, you need to generate Tokens for users to purchase:

  1. Generate Tokens in Cogmate
  2. Find your Agent in the CogNexus Dashboard
  3. Click "Manage Tokens"
  4. Add the generated Tokens
  5. Set the unit price

When a user purchases a Token, it is marked as sold and you receive the corresponding ATP.
